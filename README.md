# Plugin Osuny : centered layout

Ce plugin ajoute la gestion des alumni publics dans un site Web.

## Utilisation

D'abord, importer le plugin dans le site.

```bash
git submodule add git@github.com:osunyorg/osuny-plugin-alumni-site.git themes/osuny-plugin-alumni-site
```
